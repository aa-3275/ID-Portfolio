# Alok Singh Portfolio

NPM-based portfolio site built with Vite.

## Folder Structure

- `index.html` - main page
- `src/styles.css` - theme, layout, transitions, and responsive styles
- `src/script.js` - smooth scrolling, profile image fallback, and reveal animations
- `public/images/` - public image assets
- `public/documents/` - public PDF and document assets

## Run Locally

```bash
npm install
npm run dev
```

## Add Your Photo

Place your profile image here:

```text
public/images/profile.jpg
```
